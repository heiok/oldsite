Could not perform this operation because the default mail client is not properly installed

系统维护维护日志软件故障

Could not perform this operation because the default mail client is not properly installed

据字面意思大概就是说：无法执行此操作，因为默认的邮件客户端安装不正确。

 

每当我在Outlook Express邮件客户端点击了带链接的邮件地址或其他带链接的账号，windows xp sp3便会出现一阵假死状态，然后弹出以上提示，点击OK后，接着弹出60个微软的IE窗口，窗口内容显示Action canceled，地址栏里显示mailto:后面接着的就是我在客户端里点击的那个链接。如图：

弹出如此多窗口，也许又有人会说中毒了之类的，我可从来不会给病毒乱扣帽子，有时真替病毒感觉到很冤，大多人都将自己无法解决或无法理解的事情归究于病毒，冤！更多时候，请想想自己究竟在此之前都做了些什么吧。

我很少使用IE浏览器，一直用Firefox，所以我也没对IE作过什么设置。根据以上提示，应是由IE点击链接后调用默认邮件客户端不成功所致。查看了一下IE的设置，在Programs那项，Internet programs下的E-mail:那一栏为空，偶选了Outlook express。之后再测试点击Outlook express里的链接，就自然弹出写邮件的新窗口了。

就这样简单~~~
