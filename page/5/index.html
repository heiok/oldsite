微软提供了一个实用工具DevCon，使用 DevCon，您可以启用、禁用、重新启动、更新、删除和查询单个设备或一组设备。可以将 DevCon 用于 Microsoft Windows 2000、Windows XP 和 Windows Server 2003，但不能将其用于 Windows 95、Windows 98 或 Windows Millennium Edition。
