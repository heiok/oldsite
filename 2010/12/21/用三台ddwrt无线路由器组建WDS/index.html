用三台ddwrt无线路由器组建WDS

WDS（Wireless Disttibution System）想必各位早有所闻，因为目前WDS还没有一个公认的相关标准，现在的路由器厂商也会在自家产品里添加WDS功能，我这里用的是DD-WRT里面的WDS，其他系列请参阅对应的路由器使用说明。图片不上了，网上一搜一大把，主要说说步骤更注意事项。本例没有启用WDS无线加密！

所用三台路由器情况：

  • Router Name: client1
  • Router Model: Linksys WRT54G v4
  • Firmware Version: DD-WRT v24-sp2 (10/10/09) vpn - build 13064
  • Router IP: 192.168.8.1 2.
  • Router Name: client2
  • Router Model: Linksys WRT54GS v.3 (PCCW版）
  • Firmware Version: DD-WRT v24-sp2 (10/10/09) vpn - build 13064
  • Router IP: 192.168.8.23.
  • Router Name: client3
  • Router Model: D-Link DIR-605 B1版 （刷了后会显示D版）
  • Firmware Version: DD-WRT v24-sp2 (04/23/10) std - build 14311

  • Router IP: 192.168.8.3三台路由器的设置情况：（以client1路由器作为 主路由器）
    1. client1路由设置Setup → Basic Setup (设置 → 基本设置) WAN Setup

  • WAN Connection Type 根据自己情况选择类型并设置好；

    • Network Setup 

      *   <span style="color: #000000;">Local IP Address: 192.168.8.1</span>
      • Subnet Mask: 255.255.255.0
      • Gateway, Local DNS 0.0.0.0
        Network Address Server Settings
    • DHCP Type: DHCP Server
    • DHCP Server: Enable
    • Start IP Address: 192.168.8.100 (分配地址不要包括三台路由器本身的地址）
    Wireless → Basic Settings (无线 → 基本设置)

Basic Settings

*   <span style="color: #000000;">Wireless Mode: AP </span>
*   <span style="color: #000000;">Wireless Network Mode: Mixed </span>
*   <span style="color: #000000;">Wireless Network Name (SSID): openclient </span>
*   <span style="color: #000000;">Wireless Channel: 8 – 2.447 GHz </span>
*   <span style="color: #000000;">Wireless SSID Broadca*st: Enable</span><span style="color: #000000;">WDS Settings</span>

要实现WDS，第一个下拉选择框选 LAN ，之后的六个框输入要和主路由器实现WDS功能客户端路由器的无线MAC地址，比如这里有两个，那就填入client2与client3的无线MAC地址。
2. client2路由器设置Setup → Basic Setup (设置 → 基本设置) WAN Setup

3. client3路由器设置除Local IP Address: 192.168.8.3与client2设置不同，其他设置与client2相同！

4. 所有设置完成后，保存后重启路由器就可以测试了~~~

注：

a.如果启动无线信号加密，各路由器的无线加密类型 及 密码要相同；

b.各路由器的管理登录用户名与密码不必一致。

c.如果出现无法连接上的情况，请将SPI Firewall关闭。
